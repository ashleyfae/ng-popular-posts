== Description ==

Simple widget for displaying popular blog posts.

== Installation ==

1. Upload `ng-popular-posts` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Appearance -> Widgets
1. Add the NG Social Share widget to your sidebar
